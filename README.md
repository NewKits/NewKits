# NewKits

A NewKits é um projeto web simulando um E-commerce de roupas do estilo urbano "streetwear".
O projeto foi feito sob orientação do prof. Wellington Cidade, para a demonstração das habilidades e conhecimentos adquiridos no 1º ano do curso técnico ao 1º ano de Desenvolvimento de Sistemas.
